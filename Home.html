¿Qué es TICs?

Es una carrera enfocada en el diseño, desarrollo, administración e implementación de soluciones tecnológicas. Forma profesionales capaces de integrar software, hardware y redes para resolver problemas reales.

Lo que aprenderás

Áreas Principales

Conoce las especialidades que dominan nuestros egresados

Desarrollo de Software

Aprende a crear software, aplicaciones web y móviles.

Materias relacionadas:

  • Fundamentos de Programación
  • Programación Orientada a Objetos
  • Programación Web
  • Desarrollo de Aplicaciones Móviles
  • Tópicos de Desarrollo de Sistemas

Redes y Comunicaciones

Construye, administra y asegura redes profesionales.

Materias relacionadas:

  • Fundamentos de Redes
  • Redes de Computadoras
  • Telecomunicaciones
  • Tecnologías Inalámbricas
  • Redes Emergentes
  • Administración y Seguridad de Redes

Datos, Big Data y Ciencia de Datos

Trabaja con bases de datos avanzadas y análisis inteligente.

Materias relacionadas:

  • Fundamentos de Bases de Datos
  • Taller de Bases de Datos
  • Bases de Datos Distribuidas
  • Bases de Datos NoSQL
  • Minería de Datos
  • Big Data
  • Programación para Ciencia de Datos

Arquitectura y Sistemas

Comprende cómo funciona el hardware y los sistemas operativos.

Materias relacionadas:

  • Arquitectura de Computadoras
  • Sistemas Operativos I
  • Sistemas Operativos II
  • Análisis de Señales

Ingeniería de Software y Gestión

Desarrolla proyectos profesionales y bien documentados.

Materias relacionadas:

  • Ingeniería de Software
  • Taller de Ingeniería de Software
  • Administración de Proyectos
  • Interacción Humano-Computadora
